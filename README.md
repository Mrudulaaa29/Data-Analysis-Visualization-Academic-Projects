# Data-Analysis-Visualization-Academic-Projects
Collection of academic projects and practical implementations focused on Data Analysis, Exploratory Data Analysis (EDA), Data Visualization, and Machine Learning using Python, Pandas, NumPy, Matplotlib, Seaborn, and Kaggle datasets.
Topics Covered:
• Data Analysis
• Data Visualization
• Exploratory Data Analysis (EDA)
• Linear Regression
• Logistic Regression
• Naive Bayes
• Iris Dataset Analysis
• Titanic Dataset Analysis

Tools & Technologies:
• Python
• Pandas
• NumPy
• Matplotlib
• Seaborn
• Jupyter Notebook
